### projects
- [deltarune song ranker](https://stavros-alt.github.io/drSongRanker/): rate and rank every deltarune song head-to-head. your picks get compared against everyone else's on a global leaderboard.
- [undertale-box-bot](https://github.com/Stavros-alt/undertale-box-bot): discord bot that generates undertale/deltarune-style text boxes for any character or sprite in the series.
- [steam mod downloader](https://github.com/Stavros-alt/steamModDownloader): downloads steam workshop mods without steamcmd. scrapes smods.ru via selenium. supports batch lists and headless mode.
